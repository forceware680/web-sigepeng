## Topik Lanjutan

### State Management

Mengelola state dalam aplikasi adalah skill penting:

- **Local State** - State yang hanya ada di satu komponen
- **Global State** - State yang bisa diakses dari mana saja
- **Server State** - Data yang berasal dari server

### Optimasi Performa

Beberapa teknik untuk mengoptimalkan performa aplikasi:

1. **Lazy Loading** - Memuat komponen hanya saat diperlukan
2. **Caching** - Menyimpan data yang sering diakses
3. **Code Splitting** - Memecah kode menjadi bagian-bagian kecil

### Deployment

Langkah-langkah untuk deploy aplikasi:

1. Build project untuk production
2. Pilih platform deployment (Vercel recommended!)
3. Configure environment variables
4. Deploy dan monitor
